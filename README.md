# DNN PROJECT

# How to run the code:
1. Clone the repository:
    ``` git clone https://github.com/RishiTez/dnn-proj.git ```
2. Install dependencies:
    ``` pip install -r requirements.txt ```
3. Run any of the models:
    ``` python3 FNN.py ```
   ``` python3 RNN.py```
    ``` python3 CNN.py ```
